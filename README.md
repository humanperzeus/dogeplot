# 📊 DOGEPLOT - Revolutionary Congressional Bill Tracker

**Transforming how we track, analyze, and understand legislation through AI-powered intelligence**

![DOGEPLOT Hero Image](https://dogeplot.fun/doge-icon.png)

## 🚀 Vision

DOGEPLOT revolutionizes how citizens, researchers, and organizations interact with congressional legislation. By combining cutting-edge AI with a powerful data platform, we're creating unprecedented transparency and insight into the legislative process.

**The Problem**: Tracking and understanding congressional bills is traditionally complex, time-consuming, and requires specialized knowledge of legislative procedures.

**Our Solution**: A lightning-fast, AI-powered platform that transforms raw legislative data into actionable intelligence through sophisticated analysis and an intuitive interface.

## 💫 Core Capabilities

### Next-Generation Intelligence
- **AI-Powered Bill Analysis**: Automatic extraction of key provisions, implications, and summaries
- **Semantic Search Engine**: Find relevant legislation based on concepts and meaning, not just keywords
- **Similarity Detection**: Discover related bills through advanced vector embeddings
- **Contextual Understanding**: Advanced models that comprehend legislative language and structure

### Comprehensive Tracking
- **Real-Time Updates**: Synchronization with Congress.gov for the latest bill status
- **Complete Bill Information**: Full text, sponsors, committees, status timelines, and voting records
- **PDF Integration**: Native support for official bill documents with text extraction
- **Historical Archives**: Access to bills across multiple congressional sessions

### Exceptional User Experience
- **Intuitive Interface**: Modern, responsive design that works across all devices
- **Powerful Filtering**: Quickly narrow down bills by date, status, type, sponsor, and more
- **Detailed Visualization**: Interactive status timelines and relationship graphs
- **Dark/Light Modes**: Optimized viewing experience for any environment

## ⚡ Technology Stack

DOGEPLOT leverages a modern, scalable architecture:

- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **Backend**: Express, Supabase, Vector Embeddings
- **AI/ML**: State-of-the-art Large Language Models, Vector Embedding Models
- **Infrastructure**: Cloud-native, serverless design with Google Cloud Run
- **Data Sources**: Direct integration with Congress.gov API

## 🛠️ Quick Start

### Prerequisites
- Node.js 18+ and npm
- Supabase account
- API keys for Congress.gov and AI services

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/dogeplot.git
   cd dogeplot
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure environment**
   ```bash
   cp .env.example .env
   # Edit .env with your credentials
   ```

4. **Launch development environment**
   ```bash
   npm run dev
   # OR use our comprehensive menu system
   npm run menu
   ```
